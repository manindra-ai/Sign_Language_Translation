# Sign Language Translation System

## 📌 Project Description
This project is designed to recognize and translate sign language gestures into text using computer vision and machine learning techniques.

## 🚀 Features
- Real-time sign language detection
- Dataset collection using webcam
- Model training for gesture recognition
- Live translation output

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Machine Learning

## 📂 Project Files
- capture_dataset.py → Used to collect gesture data
- train_model.py → Used to train the model
- real_time_translation.py → Real-time prediction

## ⚠️ Note
Due to GitHub file size limitations:
- Dataset folder is not uploaded
- sign_model.pkl is not uploaded

## ▶️ How to Run
1. Run capture_dataset.py to collect dataset
2. Run train_model.py to train the model
3. Run real_time_translation.py to test the system
