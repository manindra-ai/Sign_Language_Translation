# Sign Language Translation Project

## Files Included
- capture_dataset.py
- train_model.py
- real_time_translation.py

## Note
Due to GitHub file size limitations, the dataset and trained model file (sign_model.pkl) are not uploaded.

## How to Run
1. Run capture_dataset.py to collect data
2. Run train_model.py to train the model
3. Run real_time_translation.py for prediction

## Requirements
- Python
- OpenCV
- NumPy
- TensorFlow
- Scikit-learn
